1、JavaScript严格区分大小写，每个语句以；结束
2、注释方式  //   或者/*   */

数据类型：
	1、JavaScript不区分整数和浮点数 ，统一用Number来表示
		NaN //表示not a Number 当无法计算结果时用NaN表示
		Infinity //表示无限大，当数值超过JavaScript的Number所能表示的最大值时，就表示Infinity
	2、字符串是以单引号或双引号括起来的任意文本。比如“xyz” , 'abc'
	3、要特别注意相等运算符==。JavaScript在设计时，有两种比较运算符：
		第一种是==比较，它会自动转换数据类型再比较，很多时候会得到非常诡异的结果；
		第二种是===比较，它不会自动转换数据类型，如果数据类型不一致返回false，如果一致在比较。
		由于JavaScript这个设计缺陷 ，不要使用==比较，坚持使用===比较。
		另一例外是NaN这个特殊的Number与所有其他值都不相等，包括它自己。
		惟一能判断NaN的方法是通过isNaN()函数：isNaN(NaN);
		注意浮点数的比较，浮点数在计算过程中会产生误差，因为计算机无法精确表示无限循环小数。要比较两个浮点数是否相等，只能计算他们之差的绝对值，看是否小于某个阈值。
	4、null表示一个空的值，它和0以及空字符串不同，0是一个数值，''表示长度为0的字符串
		而null表示空，undefined表示值为定义。大多数情况下我们都应该用null。undefined仅仅在判断函数参数是否传递的情况下有用。
	5、数组是一组按顺序排列的集合，集合的每个值称为元素。JavaScript的数组可以包括任意数据
		类型，例如：[1,2,3.14,'hello',null,true]
		数组用[]表示，元素之间用,分隔。
		另一种创建数组的方法是通过Array()函数实现的，new Array(1,2,3);
		然而出于代码的可读性考虑，强烈建议直接使用[],数组的索引起始值为0.
	6、JavaScript的对象是一组由键值组成的无序集合
		var person = {
			name : 'bob',
			age : 20,
			tags : ['js','web','mobile'],
			city : 'beijing',
			hasCar : true,
			zipcode : null
		};
		JavaScript对象的键都是字符串类型的，值可以是任意数据类型。
		要获取一个对象的属性，我们用 对象变量.属性名 的方式
	7、变量在JavaScript中就是用一个变量名表示，变量名是大小写英文、数字、$和_的组合，
		且不能以数字开头。变量名也不能是JavaScript的关键字。申明变量用var语句，
		var s_007 = '007';
		可以把任意数据类型赋值给变量，同意变量可以反复赋值，而且可以是不同类型的变量，
		但是要注意只能用var申明一次。
		这种变量本身类型不固定的语言称之为动态语言，与之对应的是静态语言。静态语言在定义变量时必须指定变量类型，如果赋值的时候类型不匹配就会报错。
	8、JavaScript在设计之初为了方便初学者学习，并不强制要求用var声明变量。如果一个变量
		没用通过var声明就被使用，那么该变量就自动被声明为全局变量。
		在strict模式下运行的JavaScript代码，强制通过var声明变量，未使用var声明变量就使用的，将导致运行错误。
		启用strict模式的方法是在JavaScript代码的第一行写上：'use strict'
